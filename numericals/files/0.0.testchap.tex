\defn{Compact set}{A set $K$ in a topological space $X,\TT_X$ is compact if for every open cover $\UU$ of $K$, there exists a finite subcover $\UU'$ of $\UU$ that also covers $K$.}
\exm{The closed interval $[0,1]$ in the real numbers $\mathbb{R}$ with the standard topology is a compact set. This is because any open cover of $[0,1]$ can be reduced to a finite subcover that still covers the entire interval.}{}
